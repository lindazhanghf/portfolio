GBAM Music Wristband

GBAM Music Wristband

A wearable music instrument

GBAM stands for "Gesture Based Audio Mixing". This wearable device uses arm gestures to manipulate audio. It is geared towards DJs and performers seeking a more interactive experience.

This device is built upon Arduino and MPU-6050 accelerometer sensor.


Fairy Tale Map

A data visualization project

In this project, I visualized data from Digital Public Library of America (DPLA). The program searches the database of DPLA with the keyword “fairy tale” and various animal names, then analyze the result based on the occurance of animal names and plot the data on a map.

Visualized in Processing.
